## 🎯 프로젝트 개요

**문제점:**
- 매일 30개 업체에 엑셀 파일을 카톡으로 일일이 전송
- 업체에서 보낸 송장번호를 수기로 복사-붙여넣기
- 단순 반복 작업에 하루 1-2시간 소요

**해결책:**
- 구글 스프레드시트 + 파이썬 자동화
- 카카오 알림톡으로 링크 전송
- 업체가 직접 시트에 송장번호 입력
- 자동으로 송장 수집 및 엑셀 생성

**효과:**
- 월 25-55시간 절감
- 휴먼 에러 방지
- 실시간 송장 현황 파악

## 📦 설치 방법

### 1. Python 환경 확인
```bash
python3 --version  # Python 3.8 이상 필요
```

### 2. 가상환경 생성 및 활성화
```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화 (macOS/Linux)
source venv/bin/activate

# 가상환경 활성화 (Windows)
venv\Scripts\activate

# 활성화되면 프롬프트 앞에 (venv)가 표시됩니다
```

### 3. 라이브러리 설치
```bash
# 가상환경이 활성화된 상태에서 실행
pip install -r requirements.txt
```

### 4. 가상환경 비활성화 (작업 완료 후)
```bash
deactivate
```

### 3. 설정 파일 준비

#### A. 환경 변수 설정
```bash
cp .env.template .env
# .env 파일을 열어서 실제 값으로 수정
```

#### B. 업체 정보 설정
```bash
cp config/vendors.json.template config/vendors.json
# vendors.json 파일을 열어서 실제 업체 정보 입력
```

#### C. 구글 API 인증키 설정
1. [Google Cloud Console](https://console.cloud.google.com) 접속
2. 새 프로젝트 생성
3. Google Sheets API 활성화
4. Google Drive API 활성화
5. 서비스 계정 생성 및 JSON 키 다운로드
6. 다운받은 파일을 `config/google_credentials.json`으로 저장

#### D. 알림톡 API 설정
```bash
cp config/alimtalk_config.json.template config/alimtalk_config.json
# alimtalk_config.json 파일을 열어서 실제 API 정보 입력
```

## 🚀 사용 방법

### Step 1: 샘플 데이터 생성 (테스트용)
```bash
cd sample_data
python3 create_sample_excel.py
```

### Step 2: 발주 자동화 실행
```bash
python3 send_orders.py
```

**실행 결과:**
- 엑셀 파일을 읽어서 업체별로 분류
- 각 업체의 구글 시트에 데이터 업로드
- 업체 사장님에게 알림톡 발송

### Step 3: 송장 수집 자동화 실행
```bash
python3 collect_invoices.py
```

**실행 결과:**
- 모든 업체 시트에서 송장번호 수집
- 유효성 검증
- 이지어드민용 엑셀 파일 생성 (`output/` 폴더)

## ⚙️ 설정 가이드

### 구글 시트 권한 설정

1. 각 업체별 구글 시트 생성
2. 파일명: `[갓샵] {업체명}_발주서`
3. 권한 설정:
   - 해당 업체 담당자에게 '편집자' 권한 부여
   - 또는 '링크가 있는 모든 사용자 편집 가능' 설정

### 시트 칼럼 구조

| 주문일자 | 주문번호 | 수취인명 | 연락처 | 주소 | 상품명 | 옵션 | 수량 | 택배사 | 송장번호 |
|---------|---------|---------|--------|------|--------|------|------|--------|---------|
| (자동)  | (자동)  | (자동)  | (자동) | (자동)| (자동) | (자동)| (자동)| (입력) | (입력) |

**중요:**
- '택배사'와 '송장번호' 칸만 편집 가능하도록 설정
- 나머지 칼럼은 시트 보호 기능으로 잠금

### 알림톡 템플릿 등록

알리고/솔라피 등의 서비스에서 아래 템플릿을 등록하세요:

```
[갓샵] 발주 알림

안녕하세요, #{업체명} 사장님.
#{날짜} 신규 주문 #{건수}건이 등록되었습니다.

아래 링크에서 주문 내역 확인 후,
'송장번호' 칸에 입력 부탁드립니다.

※ 발송 완료 후 별도 연락은 필요 없습니다.
```

## 🐛 문제 해결

### "구글 API 인증 실패" 오류
- `config/google_credentials.json` 파일 확인
- 구글 클라우드 콘솔에서 API가 활성화되었는지 확인
- 서비스 계정 이메일에 시트 편집 권한 부여 확인

### "알림톡 발송 실패" 오류
- API 키가 올바른지 확인
- 발신 번호가 등록되었는지 확인
- 템플릿 코드가 승인되었는지 확인

### "엑셀 파일을 읽을 수 없음" 오류
- 파일 경로 확인
- 엑셀 파일 형식 확인 (.xlsx)
- 파일이 열려있지 않은지 확인
